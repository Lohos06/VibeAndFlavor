# 🌍 Vibe & Flavor
Project created by Clarspinon24, yasminmfth and Lohos

# 📖 Project Description

**Vibe & Flavor** is an interactive world map that allows users to explore traditional dishes from different countries.

By clicking on a country on the map, users can discover a traditional dish associated with that country, along with cultural information such as:

- Dish name
- Description
- Dietary category (vegetarian, fish, meat, halal, etc.)
- A recipe link
- Music related to the country

The goal of the project is to combine **culture, food and music discovery and interactive exploration** through a simple and engaging web interface.

# Main Features

- Interactive world map
- Display of traditional dishes and music by country
- Dietary filters (vegetarian, fish, meat, halal)
- Continent filtering
- Dynamic data retrieved from a Supabase database

# Technologies Used

- **Next.js**
- **React**
- **TypeScript**
- **CSS Modules**
- **Supabase (database)**
- **Node.js**
